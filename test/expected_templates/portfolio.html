• Mi Página Personal

    Descripción:
    Con el objectivo de tener un proyecto para aplicar nuevas tecnologías arranqué esta idea que ya ha sufrido 4 iteraciones. Volviendose más compleja pero fácil de administrar. Comenzando con una página HTML con el contendio hardcodeado a una página donde se integran servicios y bases de datos. Este proyecto está cocibido para ser un curriculum en formato web, aunque no existen limitaciones sobre futuras aplicaciones y funcionalidades.

    • HTML Page

      Esta página es la que todo el mundo visualiza. Es la fachada bonita del sistema. Diseñado sólamente con CSS3, sumando librerias de js y jQuery para efectos. La información es recibida de una base de datos Firebase en formato json y es procesada para mostrar en cada seccion los datos correspondientes. Por el momento sólo admite lenguaje español pero los servicios proveen soporte multiidioma, lo cual en las próximas etapas el sitio podría contar con esta funcionalidad. A nivel código, se utiliza GulpJs para compilar los archivos css, html y js, contando con la opción de optimizado para reducir el tamaño final de los archivos. Sumando a esta reducción del tamaño de la página, las imágenes son obtenidas de un S3 por lo caul el sitio es sólamente la estructura y estética de la página, siendo idependiente del contenido. Esta independencia de la información permite poder armarlo como plantilla para la aplicación en otros sitios. Si a eso le sumamos que el proyecto es dockerizado la practicidad de deployarlo en difentes servidores o realizar actualizaciones de versiones es extremandamente alta.
